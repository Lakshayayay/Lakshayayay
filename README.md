My **core stack spans the full MERN spectrum** — from React and Next.js on the frontend to Node.js, Express, and Python on the backend — paired with hands-on experience integrating **AI/ML APIs, LLMs, and automation pipelines**. I thrive on **problem-solving, rapid prototyping, and continuous learning**, always pushing the boundary between what's built and what's possible.
